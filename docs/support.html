Questions fréquentes

Comment importer un fichier GPX ?

Depuis l'écran principal, appuyez sur le bouton d'import. Vous pouvez sélectionner un fichier GPX depuis vos fichiers, iCloud, ou toute autre source. Les fichiers partagés depuis d'autres apps (Strava, Komoot, OpenRunner) sont également pris en charge.

Comment ajouter un jalon sur mon parcours ?

Ouvrez votre parcours et affichez le profil altimétrique. Touchez le graphique à l'endroit où vous souhaitez placer un jalon. Une fenêtre apparaît pour personnaliser le type et le message vocal.

Le guidage vocal fonctionne-t-il en arrière-plan ?

Oui. Une fois le guidage lancé, vous pouvez verrouiller votre téléphone et le ranger. Les annonces vocales se déclenchent automatiquement quand vous atteignez un jalon, même si l'écran est éteint.

L'app fonctionne-t-elle sans connexion internet ?

Oui. Une fois votre parcours importé, aucune connexion n'est nécessaire. Le guidage GPS et les annonces vocales fonctionnent entièrement hors ligne.

Pourquoi le GPS ne fonctionne pas ?

Vérifiez que vous avez autorisé l'accès à la localisation dans Réglages > Confidentialité > Service de localisation > TrailMark. Pour un guidage en arrière-plan, sélectionnez "Toujours".

Les annonces coupent ma musique, comment régler ça ?

Les annonces vocales sont conçues pour baisser temporairement le volume de votre musique puis le restaurer. Si le comportement ne vous convient pas, vérifiez les réglages audio de votre téléphone.

Compatibilité

  • Localisation GPS requise
  • Compatible avec les écouteurs Bluetooth
