## Positionnement

PaceMark est un **coach de stratégie de course**. L'app analyse un GPX, détecte les segments (montée, descente, plat), place des repères stratégiques avec des conseils adaptés, et guide vocalement le coureur pendant la course.

Ce n'est PAS :
- Un tracker de course (la montre GPS fait ça)
- Un plan d'entraînement (pas de programme de semaines)
- Une app de navigation turn-by-turn

C'est :
- Un briefing de course intelligent
- Un coach vocal qui te rappelle ta stratégie au bon moment
- Un outil de préparation de la performance le jour J

## Ton & personnalité

- **Principal** : Performance, confiance, expertise
- **Style** : Coach exigeant mais bienveillant — direct, motivant, crédible
- **Vocabulaire** : D+, relance, gestion, segment, ravito, technique, allure
- **Pas de** : bullshit motivationnel creux, ton commercial, excès d'emojis

## Cible

- Trail runners (trail court a ultra) — cible principale
- Coureurs sur route / marathon — cible secondaire
- Cyclistes (courses velo) — cible tertiaire
- Orientes performance, préparent leurs courses

## Sport principal : Trail
L'app fonctionne sur route et velo mais le trail est le coeur du positionnement.
Le nom PaceMark est volontairement multi-sport (pace = allure, mark = repère).
